# Проект: Telegram-бот для обмена валют с автоматическим парсингом и Docker-инфраструктурой

**Ноябрь 2025 | Фриланс**

- Спроектировал архитектуру сервиса из нескольких контейнеров (Bot + Selenium + MongoDB) с использованием Docker + docker-compose, обеспечив изоляцию сервисов и удобный деплой.
- Развернул Selenium в отдельном контейнере с оптимизацией ресурсов, увеличением SHM, отключением лишних сервисов и автоматической очисткой зависших процессов.
- Настроил headless Chrome с антидетект-конфигурацией и оптимизацией загрузки для снижения вероятности блокировок и повышения производительности.
- Реализовал отказоустойчивый механизм парсинга с ожиданиями, обработкой динамического контента, исключений и retry-логикой.
- Интегрировал синхронный Selenium в асинхронный Telegram-бот на базе aiogram, предотвратив блокировку цикла событий.
- Настроил задания планировщика для обновления курсов, уведомлений и повторных попыток с экспоненциальной задержкой.

### Стек
`Python`, `aiogram`, `Selenium (Remote WebDriver)`, `Docker`, `docker-compose`, `MongoDB`, `APScheduler`, `asyncio`
